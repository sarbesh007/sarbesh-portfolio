Smart Brain Face Detection (Full stack)

The smart brain face detection app detects a face in the image provided to it. The user can sign-in or register using forms and automatically gets redirected to the homepage where the user’s name and current entry count is also available. The app is responsive and compatible with various devices. The app uses AI provided by the Clarifai api Face_Detect_Model. It uses REST API to get data. The front end of the app is built with React.js, Tachyons, particles.js and react-tilt npm packages. The server is built with node Express.js and the data is stored with the help of PostgreSQL. User’s password is encrypted using bcrypt. This is a full stack app and deployed with Heroku cloud platform.

Corona Virus Tracker

This is s simple COVID-19 tracker website. With the rising cases of COVID-19 it is important to be updated with the current conditions worldwide. The website uses REST API and is built with React.js, hooks, Material-UI, chart.js, count-up, etc. It displays total cases, recoveries and deaths. The user can select individual countries from the drop-down list and view stats of that country along with a animated bar chart. The app is deployed with Github pages.
